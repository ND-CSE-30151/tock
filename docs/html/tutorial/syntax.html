tock.syntax¶

class tock.syntax.Operator¶
class tock.syntax.Symbol¶
tock.syntax.parse_multiple(s, f, values=None)¶

Parse one or more comma-separated elements, each of which is parsed using function f.

tock.syntax.string_to_config(s)¶

s is a comma-separated list of stores.

tock.syntax.string_to_configs(s)¶

s is a string in one of the following formats: - x,y - (x,y) - {x,y} - {(w,x),(y,z)} In any case, returns a set of tuples of stores.

tock.syntax.string_to_state(s)¶

s is a string possibly preceded by > or @.

tock.syntax.string_to_transition(s)¶

s is a string of the form a,b or a,b->c,d
